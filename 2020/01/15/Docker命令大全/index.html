容器生命周期

docker run

Code

使用docker run启动容器，docker在后台的标准操作流程：

  1. 检查本地是否存在指定的镜像，若不存在则从公有仓库下载。
  2. 使用镜像创建并启动容器
  3. 分配一个文件系统，并在只读的镜像层外面挂载一层可读可写层。
  4. 从宿主机配置的网桥接口中桥接一个虚拟机接口到容器中去
  5. 从地址池分配一个IP地址给容器
  6. 执行用户指定的应用程序
  7. 执行完毕后容器被终止
